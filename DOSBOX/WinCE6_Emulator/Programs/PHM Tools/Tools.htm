PHM Suspend PowerToy

The Suspend PowerToy is a small application without any user interface that simply suspend your device as the power button would.

It can assigned to a hardware button if you don't like the location of the power button on your device.

It can also be used from several applications like task switchers if they provide a mean to link to another application and don't provide a suspend function by themselves.

PHM Reset PowerToy

The Reset PowerToy is a small application enabling you to perform a soft reset (reboot) of your device as the reset button would.

It will display a confirmation dialog as a reset will lost any information you were working on in open application that have not been saved to the filesystem.
Your Program Memory and storage cards are however safe, except on devices that clear the RAM on soft resets.

You can run it in silent mode to reset without confirmation using "reset.exe /s".

PHM Desktop PowerToy

The Desktop PowerToy is another small application without any user interface that simply return to the Desktop, minimizing all windows.

It can be used from several applications like task switchers if they provide a mean to link to another application and don't provide a desktop function by themselves.

PHM Fonts applet PowerToy

Control panel applet that gets displayed in Start > Settings > Control Panel.

PHM Databases applet PowerToy

Control panel applet that gets displayed in Start > Settings > Control Panel.
Enable you to list and clean your system databases.

PHM Scheduled Tasks applet PowerToy

Control panel applet that gets displayed in Start > Settings > Control Panel.
Enable you to list and delete scheduled tasks.

PHM Fonts

The Fonts control panel applet displays your installed fonts as recognized by the operating system.
Tap a font name to preview it.

PHM Fonts Viewer

The Fonts control panel applet displays your installed fonts as recognized by the operating system.
Tap a font name to preview it.

PHM Databases

The Databases control panel applet displays your system databases.
Select a database, then select Delete in the File menu to delete it.

PHM Scheduled Tasks

The Scheduled Tasks control panel applet displays your scheduled system tasks.
Double tap a notification to display its properties, including type of schedule, triggering event and time, and notification information.
Select a notification, then select Delete in the File menu to delete it.

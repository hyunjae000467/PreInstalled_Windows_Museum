Device Manager Overview

Device Manager provides you with a graphical view of the hardware that is installed on your Pocket PC, this can be built-in devices, PC/CF Cards and external modems.
Using Device Manager, you can:

  • Determine whether the hardware on your Pocket PC is working properly.
  • Identify the device drivers that are loaded for each device.
Because the manufacturer provide the device with the corresponding hardware abstraction layer, there is nothing you can configure, but you can see the structure, check devices status and drivers.

View devices

Device Manager display devices by the type of device installed, such as infrared or modem.
Select a device and tap the properties button to display more information.
Press the refresh button to reload devices information, this is needed to update the display when you removed or inserted a device.

Windows CE assign a name to most devices, like DSK1: for the first disk device, see the devices names page for a complete description.

Devices names

SIP Soft Input Panel
This is the IME software providing the on-screen keyboard, character recognizer, or another installed input method.
COM Communication (Serial) device
Can be a serial port, an infrared adapter, a modem, or a serial interface to another device.
WAV Waveform audio device
This is a digital audio sound device.
WAM Wave API Manager
This is a driver that allow applications to generate waveform sounds.
ACM Audio Compression Manager
A codec device that simplify audio compression/decompression.
DSK Disk device
The main filesystem, called the Object Store, is in RAM. Disk devices are aditionnal storage devices, a CompactFlash memory card for example.
NDS NDIS device
A network interface adapter, allow you to connect to a network.

External modems don't have a device name, they just link a modem driver to an existing COM port.
PWO (PowerOn) and RCN (remocon) are Cassiopeia specific devices.

Device Properties

Type Describe the function of the device.
Registry Driver definition location in the system registry.
Driver files Device Driver : The driver file used for this device.
TSP driver : Transport Service Provider for this device.
File System : File System driver for this device.
Resources Device ID : Windows CE ID for this device.
Socket : PCMCIA Socket used by this device.
Plug and Play ID Plug and Play identification string, built-in devices usually does not provide this.

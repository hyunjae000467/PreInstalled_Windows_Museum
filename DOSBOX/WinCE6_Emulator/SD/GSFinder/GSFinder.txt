●　動作環境、および必要なDLL
・WindowsCE2.11日本語版以上を搭載したHandheld PC Pro. 3.0
・WindowsCE3.0日本語版以上を搭載したHandheld PC 2000
・NTT Docomo SigmarionIII

●　インストール
・インストーラーはありません。
・解凍する際はフォルダ付きで解凍してください。
・解凍後作成されるexeファイルをHPCにコピーして実行してください。
・使用しているHPCに適合したものを使用してください。
MIPS CPU版:	MIPSフォルダ
SH3 CPU版:	SH3フォルダ
SH4 CPU版:	SH4フォルダ
ARM CPU版:	ARMフォルダ
SigmarionIII版:	Sig3フォルダ

●　動作確認
NTT Docomo sigmarionIIIとHP jornada720で動作確認を行っております。

●　基本機能
PocketPC用のファイラであるGSFinderをHPCでも動作するようにしたものです。
HPCを基準に作成したものではないので若干使いにくいところがあると思われます。
Handheld PCのエクスプローラより優れている点は以下の通りです。
・フォルダのツリー表示とファイルリストの2画面構成となっています。
・ファイルのプロパティを強化してあります。
・シングルクリックで操作を行うことが可能です。
・PocketPCと同様のタップアンドホールドでポップアップメニューが開けます。


●　操作方法
・基本的な操作方法はWindowsのエクスプローラと同じですので省略します。
・”名前をコピー”では選択中のファイル、フォルダの名前をクリップボードにコピーします。
・”パスをコピー”では選択中のファイル、フォルダのフルパス（位置情報）を
クリップボードにコピーします。
・パラメータ付きで実行では、選択中の実行時のパラメータを指定して実行できます。
Exeファイルに開かせたいファイルのパスをパラメータとして指定して実行することができます。
・ファイルのプロパティ表示時に複数のファイルが選択されていると、
ファイルのサイズを計算して表示するようになっています。
・表示メニューの”シングルクリック操作”をチェックすると
シングルクリックでファイルを開けるようになります。
・指定のフォルダにあるEXEファイル、EXEファイルのショートカットを読み込み、
”ファイル”メニュー→”開く”に追加する機能を実装しました。
”開く”メニューにアプリケーション名が表示されますが、
アプリケーションにファイルのパスをパラメータとして渡すことにより、
指定のアプリケーションでファイルを開くことができます。


●　オプション設定
”表示”メニューのオプション設定を変更することによりGSFinderをさらに便利に使うことができます。
・”表示しないファイル”を設定することにより、ファイル一覧の読み込みを制限し動作を
高速にすることができます。次のファイルの読み込みを制限することができます。
	隠しファイル
	ROMファイル
	システムファイル
・”前回終了したときのフォルダ...読み込む”では、終了時に表示していたフォルダを保存し、
再起動時にそのフォルダを表示するというものです。
・”ショートカット作成時に...つけない”では、通常はショートカット作成時に
”のショートカット”という名前を付けるようになっていますが、これをつけないようにします。
・”拡張子を表示しない”では、ファイル一覧で拡張子を表示しないようにします。
・オプション設定の”ファイルのアイコンを読み込まない”ではファイルの
拡張子別で登録されたアイコンを読み込まず、統一のファイルアイコンで表示するようにします。
ファイル一覧読み込みが高速化します。
・オプション設定の”ROMファイルの...表示する”ではシステム属性のファイルを、
統一のROMファイルアイコンで表示するようにします。ROMファイルを見分けるのに便利です。
・”アイコンキャッシュのクリア”ボタンを押し、ダイアログをOKを押すと
アイコンのキャッシュをクリアして、読み直します。
このソフトでは一度読み込んだファイルのアイコンをキャッシュとして保存して、
再読込時に高速化するようになっていますので、常に最新のアイコンを読み込んでいるわけではありません。
間違ったアイコンが表示されている場合は、アイコンキャッシュをクリアしてみてください。


●　シングルクリック操作について
ファイルを開く操作をシングルクリックでできます。
ファイルを開く操作が簡単にできるようになりましたが、複数ファイルの操作や、
ファイル名を変更する操作がしにくくなります。
表示メニューの”シングルクリック操作”をチェックすると
シングルクリックでファイルを開けるようになります。


●　キーボードショートカットについて
GSFinder for Handheld PCにはキーボードで扱いやすいように、
キーボードショートカットを割り当ててあります。
ただし、キーボードショートカットが有効になるのはファイルリストにフォーカスがある場合のみです。
以下はキーボードショートカットと機能の一覧です。

・BS (バックスペース)
一つ上の階層のフォルダに移動
・DEL、Ctrl + D
選択されているファイルを削除
・Ctrl + C
ファイルをコピー
・Ctrl + X
ファイルを切り取り
・Ctrl + V
ファイルを貼り付け
・Ctrl + A
すべてのファイルを選択
・Ctrl + F
フォルダツリーの表示/非表示切り替え
・Ctrl + R
最新の状態に更新
・Ctrl + O
指定のフォルダにファイルをコピー（フォルダ選択ダイアログが表示されます。）
・Ctrl + T
指定のフォルダにファイルを移動（フォルダ選択ダイアログが表示されます。）
・Ctrl + S
指定のフォルダにファイルのショートカットを作成（フォルダ選択ダイアログが表示されます。）
・Ctrl + L
リストの表示スタイルの変更
・Ctrl + M
ポップアップメニューの表示
（当然、ALT + タップでもポップアップメニューは表示されます。）
・Alt + Enter、Ctrl + P
プロパティの表示
・F5
最新の状態に更新
・F6
アドレスバーにフォーカス
・Ctrl + B
パラメータ付きでファイルを開く
・Ctrl + N
新規フォルダ作成
・Ctrl + Q
終了

●　注意事項
・コピー、移動、削除のダイアログはマルチスレッドになっています。
そのためキャンセルを押しやすくなっていますが、スレッドが終了するまでダイアログは閉じません。
・過去バージョンのレジストリとは互換性がありません。

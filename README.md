# PreInstalled_Windows_Museum
경기고등학교 컴퓨터부 CE-L에서 화동제에 전시했던 고전 OS를 구동했던 파일들입니다. 원하신다면 직접 구동해 볼 수도 있습니다.

- 무엇이 되는가?
  - 마우스
  - 키보드
  - OS 구동
